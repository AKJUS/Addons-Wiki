# 无尽反应堆

无尽反应堆无尽无尽科技的终极发电机之一。  
无尽反应堆需要无尽锭和虚空锭，通过两者碰撞产生电量，缺一不可。  
不同于其他反应堆，它没有任何限制条件，放置在任何连入能源网络地方都可以发电，并且可以直接接入货运而不需要反应堆访问接口。  
即使能源网络中没有机器需要电力，它也不会停止工作。  

## 等级

目前无尽反应堆有1个等级。

**效率**: 240,000 J/s  
**可储存**: 120,000,000 J

## 反应堆燃料

默认的1个sf ticks是0.5s (10 ticks)

| 类型  | 持续时间 |
| :--- | :-: |
| 无尽锭 | 196000 `粘液刻`/个 | 
| 虚空锭 | 32000 `粘液刻`/个 |
